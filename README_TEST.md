## Catatan Yang Masih Kurang

- Export laporan sudah tersedia dalam format CSV, tetapi belum ada ekspor PDF atau Excel.
- Print struk sudah mendukung mode cetak browser, tetapi belum dioptimalkan untuk layout thermal receipt.
- Notifikasi stok menipis dan stok habis belum otomatis muncul di dashboard atau topbar.
- Manajemen staff pada sidebar belum dibuat fungsional.
- Validasi dan otorisasi masih dasar, belum memakai Form Request terpisah dan policy yang lebih lengkap.
- Belum ada test khusus untuk tampilan Blade, pencarian menu, dan alur pembayaran cash/qris/debit secara lebih detail.
