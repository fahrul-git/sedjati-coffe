Codex berhasil terhubung

## Catatan Yang Masih Kurang

- Halaman login dan laporan belum disesuaikan penuh dengan desain admin panel baru.
- Fitur export laporan masih berupa tampilan tombol, belum ada proses ekspor PDF atau Excel.
- Tombol print pada struk belum dibuat ke mode cetak khusus browser atau thermal receipt.
- Upload gambar produk belum tersedia, jadi kartu menu masih memakai visual placeholder.
- Filter lanjutan untuk menu dan orders masih bersifat tampilan, belum memakai query filter backend.
- Notifikasi stok menipis dan stok habis belum otomatis muncul di dashboard atau topbar.
- Manajemen customer, settings, dan staff pada sidebar belum dibuat fungsional.
- Validasi dan otorisasi masih dasar, belum memakai Form Request terpisah dan policy yang lebih lengkap.
- Belum ada test khusus untuk tampilan Blade, pencarian menu, dan alur pembayaran cash/qris/debit secara lebih detail.
